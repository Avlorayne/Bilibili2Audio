# Bilibili2Audio 业务流程与人机交互设计文档

## 1. 概述

本文档详细描述Bilibili2Audio的完整用户操作流程、核心功能交互逻辑、异常场景处理及UI交互规范。

## 2. 完整用户操作流程

### 2.1 启动阶段流程

```
用户操作                      系统响应
─────────────────────────────────────────────────────────────────
双击 start.sh/start.bat  →   显示启动控制台窗口
                             │
                             ├── 检查Node.js
                             │   ├── 已安装 -> 显示"Node.js 版本: v18.x.x"
                             │   └── 未安装 -> 弹出提示框：
                             │       "请安装Node.js 18.0或更高版本"

                             │

                             │   ├── 已安装 -> 显示"依赖检查通过"

                             │       显示安装进度...

                             ├── 检查FFmpeg
```

#### 3.4.2 任务状态管理

```javascript
const taskStatus = {
  QUEUED: 'queued',
  PROCESSING: 'processing',
  COMPLETED: 'completed',
  FAILED: 'failed',
  CANCELLED: 'cancelled'
};

// 任务队列管理
class TaskQueue {
  constructor(maxConcurrent = 3) {
    this.maxConcurrent = maxConcurrent;
    this.queue = [];
    this.active = new Map();
  }
  
  addTask(task) {
    if (this.active.size < this.maxConcurrent) {
      this.startTask(task);
    } else {
      this.queue.push(task);
    }
  }
  
  startTask(task) {
    this.active.set(task.id, task);
    // 开始处理任务
  }
  
  completeTask(taskId) {
    this.active.delete(taskId);
    if (this.queue.length > 0) {
      this.startTask(this.queue.shift());
    }
  }
}
```

| 链接错误 | 格式不正确 | 错误 | 提供格式示例 | 可重试 |
| 链接错误 | 视频不存在 | 错误 | 检查链接 | 可重试 |
| 链接错误 | 需要登录 | 警告 | 提示登录 | 需手动处理 |
| 解析失败 | API不可用 | 错误 | 重试 | 可重试 |
| 解析失败 | 地区限制 | 警告 | 提示限制 | 不可恢复 |
| 转码失败 | 文件损坏 | 错误 | 检查文件 | 可重试 |
| 转码失败 | 空间不足 | 错误 | 清理空间 | 可恢复 |
| 转码失败 | 格式不支持 | 错误 | 更换格式 | 可重试 |
| 网络错误 | 连接超时 | 错误 | 重试 | 可重试 |
